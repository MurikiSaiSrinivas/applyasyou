# Resume content reference

The verbatim key content (summary, skills, top bullets) of each resume variant, so
the agent can reference what a resume says without re-rendering the PDF each time.

The agent fills this during onboarding (CLAUDE.md Section 1, Step 3) and appends a
section whenever it builds a new variant (the builder writes a `.content.md` sidecar
next to each resume you can paste in here).
